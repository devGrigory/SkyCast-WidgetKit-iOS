# SkyCast-WidgetKit-iOS

An iOS application built with SwiftUI and WidgetKit to showcase weather data.

## Preview
![SkyCast Widget Demo](Assets/skycast-demo.gif)

## Features
- Interactive Home Screen Widgets.
- Supports Dark and Blue theme modes.
- Built using Swift and WeatherKit.

## Requirements
- iOS 17.0+
- Xcode 15.0+
